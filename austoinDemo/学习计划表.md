## 📚 包和模块管理（必须掌握）

### 核心概念

#### 1. go.mod 和 go.sum

| 文件 | 作用 | 生成命令 |
|-----|------|---------|
| `go.mod` | 声明模块名、版本、依赖 | `go mod init 模块名` |
| `go.sum` | 依赖校验和（安全） | `go mod tidy` |

#### 2. 包调用规则

```
同一个目录 → 同一包名 → 直接调用
不同目录   → 不同包名 → 需要导入（import）
```

#### 3. 可见性规则

```go
func PublicFunc() {}    // 首字母大写 = 公开（其他包可访问）
func privateFunc() {}   // 首字母小写 = 私有（其他包不可访问）
```

#### 4. 本地包调用（使用 replace）

**根目录 go.mod**：
```go
module github.com/你的用户名/项目名

replace 远程路径 => 本地路径
```

**main.go 导入**：
```go
import 别名 "远程路径"
别名.公开函数()
```

#### 5. 远程仓库包调用

```go
import (
    "github.com/gin-gonic/gin"  // 第三方库
)

go mod tidy  // 自动下载依赖
```
